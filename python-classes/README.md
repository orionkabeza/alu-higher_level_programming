# Python - Classes and Objects

This project covers Object-Oriented Programming (OOP) in Python, focusing on classes, objects, attributes, and methods.

## Learning Objectives

- What is OOP and first-class everything
- What is a class, object, and instance
- Public, protected, and private attributes
- The __init__ method and self
- Data Abstraction, Encapsulation, and Information Hiding
- Properties vs getters and setters

## Files

| File | Description |
|------|-------------|
